# HKZC-DevOps-Tools

## 介绍

用于存放公共的运维脚本或者依赖资源，解决通用的部署问题

## 文件夹说明

* Docker: 存放部署docker-ce相关的脚本和依赖文件
* docker-build: 存放自主构建的docker镜像模板
* docker-compose.yml: 存放一些中间件或者工具的docker-compose模板
* sql: 存放一些中间件或者工具的SQL初始化脚本
* LinuxBench: 存放一些Linux性能测试工具
